Resale Business Automation

AI-assisted eBay listing tool for clothing resellers.

Overview

Resale Business Automation is a personal reseller tool that accelerates creating high-quality eBay listings from photos. It reads photos of a clothing item, extracts the brand, size, color, material, condition, and item-specific details using AI vision, and creates a draft listing on the seller's eBay account via the eBay Inventory API.

How it works

1. Drop photos

The seller drops a batch of photos taken on their phone. The tool auto-groups photos into individual items using EXIF timestamps, so a day's worth of inventory can be prepared in one go.

2. AI analysis

Claude AI analyzes each item's photos in a two-stage pipeline — a fast broad-coverage pass, then a high-accuracy verification of the silhouette, title, and key attributes. The tool also identifies which photo is the full garment view (the photo buyers see first) vs. which shows the size/brand tag.

3. Review & edit

Each item is shown as a card with the AI's proposed title, price, condition, and item-specific fields. The seller can edit any field, fix mistakes, or remove items before anything is sent to eBay.

4. Create listings

The tool uploads photos to eBay Picture Services, creates inventory items and unpublished offers via the Inventory API, and — on the seller's click — publishes each offer as a live eBay listing.

APIs used

  • Sell Inventory API — inventory item creation, offer creation, and publishing.
  • Sell Account API — read the seller's business policies (shipping, payment, return).
  • Trading API (UploadSiteHostedPictures) — upload photos to eBay Picture Services (EPS).
  • Commerce Taxonomy API — auto-suggest eBay category IDs based on item details.
  • Buy Marketplace Insights API — comparable sold-price research to inform pricing decisions.

Expected volume

Approximately 90 listings per day during active listing sessions, equating to roughly 500–700 API calls per day (including image uploads, inventory PUT, offer creation, policy lookups, and publishing).

Privacy

This tool is used by a single eBay seller to manage their own seller account. No third-party user data is collected or stored. The seller's OAuth refresh/access tokens are stored locally on the seller's computer and cleared immediately on disconnect. Item photos are uploaded to eBay Picture Services (eBay's own hosting infrastructure). No data is shared with any third party.

An exemption from eBay's Marketplace Account Deletion notification program is in place, reflecting the no-third-party-user-data nature of the tool.
